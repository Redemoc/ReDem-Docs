## Known Limitations

- Decipher: BAS does not support custom event mapping for matrix questions.
- Keyingress: Response payload size limited to 2 MB per callback.
- SurveyWave: Asynchronous callbacks only, no synchronous response option.
- InsightForge: CHS only for single-select and multi-select question types.
- PulseLab: API v1 only, migration to v3 required for additional checks.
