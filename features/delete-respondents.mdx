You can delete one or more respondents directly in the [ReDem application](https://app.redem.io/) without using the API.

## Steps

1. Open the survey and go to the **respondent table**.
2. Select the respondent row or rows you want to remove.
3. Click the **delete** action (available when respondents are selected).
4. Confirm in the dialog. After you confirm, the selected respondents and their data are **permanently removed** from ReDem storage.

## When to use this

This flow is useful for removing **test respondents** on API-driven projects and for any case where you need to drop specific interviews from a survey while keeping the project itself.

For programmatic deletion, see the API endpoint [Delete Respondents](/api-reference/endpoints/v3/deleteRespondents).
