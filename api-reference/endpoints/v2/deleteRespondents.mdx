---
title: 'Delete Respondents'
openapi: 'POST /v2/deleteRespondents'
---

Delete one or more respondents from a survey. Also removes related metadata, interaction data, and flattened records. If no respondents remain for the survey, the survey is deleted automatically.

To delete respondents from the [ReDem application](https://app.redem.io/) (for example test interviews), see [Delete respondents](/features/delete-respondents).
