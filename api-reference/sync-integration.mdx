<Info>To integrate ReDem with your survey tool, you'll need a ReDem account. <br/> Donâ€™t have an account yet? Get started by [setting up your account](/account-setup).</Info>

<Steps titleSize="h3">
  <Step title="Login and create an API key">
    <p> Login into your ReDem account and create an API key. </p>

    1. Login to ReDem application and navigate to the account
    2. Create new API key and copy the API Key
  </Step>
  <Step title="Prepare your request data">
    <p>Ensure your data is formatted correctly according to the guidelines before send via API</p>

    ```javascript Example request body
    {
      "respondentId": "ID",
      "surveyName": "String",
      "datapoints": [
        {
          "questionId": "Q1",
          "qualityCheck": "OES",
          "openEndedAnswer": "ABC",
          "title": "XYZ",
          "keywords": ["XYZ", "ABC"],
          "isDuplicateEnabled": true
        },
        {
          "questionId": "Q1",
          "qualityCheck": "TS",
          "duration": 40
        },
        {
          "questionId": "Q2",
          "qualityCheck": "GQS",
          "itemBattery": [7, 8, 9, 0, 3, 5, 6]
        },
        {
          "questionId": "CHS_Question",
          "qualityCheck": "CHS",
          "interview": [
            { "questionTitle" : "XYZ", answer: "ABC" },
            { "questionTitle" : "abc", answer: "1" },
          ]
        }
      ],
      "needRealtimeResponse": false,
      "cleaningSettings": {},
      "reviewSettings": {}
    }
    ```
  </Step>
  <Step title="Make an API request">
    <p>Submit an API request to ReDem, including the respondent's information and the API key</p>
  </Step>
  <Step title="Interpret the API Response">
    <p>The API will respond with a JSON object contains the results of the evaluation. Review the result and identify and filter out low-quality respondents.</p>
  </Step>
  <Step title="Synchronize respondent evaluations">
    <p>Synchronize Respondent evaluations with ReDem.</p>
  </Step>
</Steps>
